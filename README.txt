Для роботи програми, файл "Parser relise version" потрібно цілком відкрити в інтегрованому
середовищі розробки "PyCharm". Відповідно "PyCharm" має бути встановлений на комп'ютері. 
Найпростіший спосіб запустити програму це натиснути правою кнопкою миші на папку "Parser relise version"
і обрати пункт "Open Folder as PyCharm Comunity Edition Project"

Перед запуском додатку потрібно встановити всі бібліотеки які використовує програма.
Для цього протібно відкрити консоль за допомогою комбінації клавіш win + r
У випадаючому вікні ввести команду cmd
В консоль потрібно ввести наступні команди:
pip install Flask
pip install requests
pip install beautifulsoup4

Коли усі бібліотеки встановлено, можна починати роботу програми. Для цього потрібно запустити
файл "main.py"

Щоб відкрити пошуковик і опинитись на веб сторінці потрібно перейти за посиланнями яке
з'явиться в консолі. Виглядатиме воно приблизно так "http://127.0.0.1:5000" і буде 
підсвічене синім кольором.

P.S. Якщо результат пошуку не в повній мірі відповідає пошуковому запиту доцільно виконати
запит ще декілька разів, доки результат не буде задовільним.
